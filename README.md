# Kitsune Sakura.exe 🌸

**Technology with a Soul** — Sitio web oficial de Kitsune Sakura.exe S.A.S.

## Stack
- HTML5 semántico
- CSS3 con variables custom (sin frameworks)
- JavaScript vanilla ES6+
- Google Fonts: Cormorant Garamond + DM Mono + DM Sans

## Características
- Bilingüe ES/EN con persistencia en localStorage
- Animaciones de scroll con IntersectionObserver
- Formulario de contacto con mailto
- Responsive (mobile, tablet, desktop)
- Sin dependencias externas

## Cómo correrlo localmente
```bash
# Solo abre index.html en tu navegador, o usa un servidor local:
npx serve .
# o
python3 -m http.server 3000
```

## Deploy en GitHub Pages
1. Sube los archivos al repo `Kitsune-sakura-exe/Kitsune-sakura-exe.github.io`
2. Ve a Settings → Pages → Source: main branch / root
3. Tu sitio estará en: `https://kitsune-sakura-exe.github.io`

## Archivos
```
/
├── index.html    — Estructura y contenido
├── styles.css    — Estilos y paleta de marca
├── main.js       — Lógica bilingüe y animaciones
└── README.md     — Este archivo
```
